# Social Media Manager - Deployment Checklist

## BACKEND DEPLOYMENT
[ ] 1. Clone repository: git clone https://github.com/HLPFLCG/socialmediamanager.git
[ ] 2. Switch to branch: git checkout fix/security-and-modernization
[ ] 3. Navigate to backend: cd cloudflare
[ ] 4. Install dependencies: npm install
[ ] 5. Login to Cloudflare: npx wrangler login
[ ] 6. Create D1 database: npx wrangler d1 create socialmediamanager-db
[ ] 7. Update wrangler.toml with database_id
[ ] 8. Initialize schema: npx wrangler d1 execute socialmediamanager-db --file=schema.sql --env production
[ ] 9. Set JWT secret: npx wrangler secret put JWT_SECRET --env production
[ ] 10. Deploy backend: npx wrangler deploy --env production
[ ] 11. Save backend URL: _______________________________________

## FRONTEND DEPLOYMENT
[ ] 1. Go to Cloudflare Dashboard: https://dash.cloudflare.com
[ ] 2. Click Pages → Create a project
[ ] 3. Connect to Git → Select GitHub
[ ] 4. Select repository: HLPFLCG/socialmediamanager
[ ] 5. Configure settings:
      - Branch: fix/security-and-modernization
      - Framework: HTML
      - Build output: frontend
[ ] 6. Add environment variables:
      - API_URL = (your backend URL from step 11 above)
      - JWT_SECRET = hlpfl-social-media-manager-jwt-secret-2024-secure
[ ] 7. Click "Save and Deploy"
[ ] 8. Save frontend URL: _______________________________________

## OAUTH SETUP - TWITTER
[ ] 1. Create app at: https://developer.twitter.com/en/portal/dashboard
[ ] 2. Enable OAuth 2.0
[ ] 3. Set callback URL: (your frontend URL)/auth/twitter/callback
[ ] 4. Get API Key and Secret
[ ] 5. Set secrets:
      npx wrangler secret put TWITTER_API_KEY --env production
      npx wrangler secret put TWITTER_API_SECRET --env production

## OAUTH SETUP - LINKEDIN
[ ] 1. Create app at: https://www.linkedin.com/developers/apps
[ ] 2. Add redirect URL: (your frontend URL)/auth/linkedin/callback
[ ] 3. Request scopes: r_liteprofile, r_emailaddress, w_member_social
[ ] 4. Get Client ID and Secret
[ ] 5. Set secrets:
      npx wrangler secret put LINKEDIN_CLIENT_ID --env production
      npx wrangler secret put LINKEDIN_CLIENT_SECRET --env production
